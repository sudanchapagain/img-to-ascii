# Image to ASCII

converts image to ASCII via decoding image to extract brightness level and
mapping brightness level to respective ASCII char.

two modes exist because the default event tho correct approach doesn't look
right due to pixels being blocks but chars being tall. `hack` flag skips every
other line to make the output correct aspect ratio to compensate for character.

the output for pepe frog is as follows
